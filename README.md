# AI极客俱乐部成员展示平台

这是一个基于GitHub Pages搭建的AI极客俱乐部成员档案展示网站，包含52位成员的技术档案信息。

## 🌐 网站功能

- **成员档案展示**：展示每位成员的基本信息、技术专长、实践案例
- **智能筛选功能**：
  - 按工作地点筛选
  - 按业务线筛选
  - 按技术标签筛选（多选）
  - 关键词搜索
- **响应式设计**：适配手机、平板、电脑等不同设备
- **成员详情查看**：点击卡片查看完整档案信息

## 📊 数据统计
- 总成员数：52人
- 工作地点：15个不同城市/办公点
- 技术标签：12种不同技术方向
- 业务线：涵盖15个不同业务领域

## 🚀 快速访问
网站已部署到GitHub Pages，访问地址：https://[你的用户名].github.io/ai-geek-club/
